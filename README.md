# Eurizon Investment Client Portal


This app was created for Eurizon Investment.
It's a Vite+React app that communicates with the Eurizon Investment API.

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

For more information and support, please contact Eurizon Investment support at support@eurizoninvestment.com.
